<h1>Como executar o projeto?</h1>

<h2>back-end:</h2>

```
1 - npm i
2 - npm run migrate 
3 - npm run seed
4 - npm start
```
**http://localhost:3000**

<h2>front-end:</h2>

```
1 - npm i
2 - npm install react-typed --force
3 - npm install sweetalert2 --force
4 - npm run dev
```
**http://localhost:5173**
